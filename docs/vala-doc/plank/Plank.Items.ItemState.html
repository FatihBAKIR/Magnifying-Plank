public enum ItemState

The current activity state of an item. The item has several states to track and can be in any combination of them.


Namespace: Plank.Items
Package: plank

Content:

Enum values:

  • NORMAL - The item is in a normal state.
  • ACTIVE - The item is currently active (a window in the group is focused).
  • URGENT - The item is currently urgent (a window in the group has the urgent flag).
  • MOVE - The item is currently moved to its new position.
